# Language App

## Описание проекта

Мобильное приложение для изучения иностранных языков, разработанное как MVP-проект с упором на удобный пользовательский интерфейс, простую навигацию и масштабируемую архитектуру.

Приложение позволяет пользователю изучать слова, проходить базовые упражнения, взаимодействовать с обучающим контентом и отслеживать прогресс обучения.

Проект реализован в формате mobile-first с использованием React Native и Expo, что позволяет быстро разрабатывать и тестировать приложение сразу под iOS, Android и Web.

---

## Назначение проекта

Основная цель проекта — создать удобное MVP-приложение для изучения языка, которое можно масштабировать в полноценную образовательную платформу.

В рамках проекта были реализованы:

* главный экран приложения
* экран уроков
* карточки слов
* базовая навигация между экранами
* адаптация под мобильные устройства
* структура для дальнейшего расширения функционала
* подготовка под добавление авторизации и личного кабинета

Проект ориентирован на дальнейшее развитие в сторону полноценного EdTech-продукта.

## Системные требования

Для запуска проекта необходимо:

* Node.js 18+
* npm или yarn
* Expo CLI
* Android Studio (для Android)
* Xcode (для iOS, только macOS)
* Expo Go (для тестирования на физическом устройстве)

## Установка и запуск проекта

### 1. Клонировать репозиторий

```bash
git clone <repo-url>
```

### 2. Перейти в папку проекта

```bash
cd language-app
```

### 3. Установить зависимости

```bash
npm install
```

### 4. Запустить проект

#### Запуск Expo

```bash
npm run start
```

#### Запуск на Android

```bash
npm run android
```

#### Запуск на iOS

```bash
npm run ios
```

#### Запуск Web-версии

```bash
npm run web
```
